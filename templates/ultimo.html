Proyecto Linea de Espera


APLICACIÓN DE UN MODELO DE TEORÍA DE COLAS O LÍNEA DE ESPERA PARA MEDIR LA OPTIMIZACIÓN EN EL SERVICIO DE ATENCIÓN AL CLIENTE EN UN LOCAL DE COMIDA

Importancia
En la actualidad en un negocio es de suma importancia no solo ofrecer un producto sino, también satisfacer las necesidades del cliente y que se sientan a gusto con el servicio que se le ofrece, la teoría de cola mediante el modelo de líneas de espera ayuda a resolver dicho problema mediante la implantación de este software y el análisis de datos, se dará una simulación de los datos estimado donde un cliente debe ser atendido

Objetivo
El sistema lo que busca es minimizar los tiempo de los clientes en el estado de cola es decir saber el tiempo simulado al demorarse al darle en servicio de esa manera el sistema ayuda que el local no pierda cliente y tengan una atención satisfactoria

¿Para que sirve?
La teoria de colas en un servicio como podemos darnos cuenta con una llegada del cliente como fuente de entrada luego cada cliente ingresa al sistema de cola donde el primero en entrar es el primero en salir donde le brinda el servicio el cliente.
